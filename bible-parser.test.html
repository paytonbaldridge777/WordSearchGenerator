📖 Bible Parser Test Suite

🔧 How to Run Tests

  1. Open this file (bible-parser.test.html) in a web browser
  2. Click the "Run All Tests" button below
  3. Review the test results - green indicates pass, red indicates failure
  4. Check the browser console for detailed logs
  5. To test with real Bible files, use the "Test Real Files" button

➕ How to Add New Test Cases

  1. Locate the <script> section at the bottom of this file
  2. Add your test data to the appropriate testData object
  3. Create a new test function following the pattern of existing tests
  4. Add your test to the runAllTests() function
  5. Refresh the page and click "Run All Tests"
