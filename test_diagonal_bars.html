🔧 Diagonal Bar Rounded Corners - Visual Test

Purpose: This test demonstrates the fix for diagonal word highlight bars, showing that they now have rounded ends matching horizontal and vertical bars.

Test: Generate Sample PDF

This will create a PDF with three types of bars to compare:

  • Horizontal bar - Uses roundedRect() with rounded corners
  • Vertical bar - Uses roundedRect() with rounded corners
  • Diagonal bar - Uses stroked line with setLineCap('round') ✨ NEW!

✅ Expected Results

All three bars should have:

  • Rounded capsule ends (circular/elliptical caps)
  • Semi-transparent grey fill (RGB 200, 200, 200)
  • Darker grey border (RGB 120, 120, 120)
  • 85% bar thickness relative to cell size
  • Consistent visual appearance

Implementation Details

Previous approach for diagonal bars:

  • Used filled polygons via doc.lines()
  • Could not achieve rounded ends
  • Sharp rectangular corners

New approach for diagonal bars:

  • Uses stroked lines with doc.line()
  • Sets doc.setLineCap('round') for circular end caps
  • Draws two layers: border (thicker) + fill (normal thickness)
  • Perfectly rounded ends matching horizontal/vertical bars
